O COMUNICA-TEA é uma aplicação mobile desenvolvida em Flutter que visa auxiliar crianças com Transtorno do Espectro Autista (TEA) na comunicação através de **pictogramas interativos** e **síntese de voz**.


### ✨ Funcionalidades

- ✅ **Pictogramas Categorizados**: Necessidades, Sentimentos, Ações e Pessoas
- ✅ **Síntese de Voz**: Áudio em português para cada pictograma
- ✅ **Formação de Frases**: Combine pictogramas para formar frases completas
- ✅ **Interface Adaptada**: Design intuitivo e cores amigáveis
- ✅ **Funciona em Múltiplas Plataformas**: Android, Linux Desktop e Web
- ✅ **Código Aberto**: Totalmente gratuito e personalizável

## 🚀 Como Executar o Projeto

### Pré-requisitos

Antes de começar, você precisa ter instalado em sua máquina:

| Ferramenta | Versão Mínima | Link para Download |
|------------|---------------|-------------------|
| Flutter SDK | 3.41.3 | [flutter.dev](https://flutter.dev/docs/get-started/install) |
| Android Studio | 2024.3+ | [developer.android.com](https://developer.android.com/studio) |
| Git | 2.25+ | [git-scm.com](https://git-scm.com/downloads) |
| VS Code (opcional) | 1.98+ | [code.visualstudio.com](https://code.visualstudio.com/) |

### 📥 Passo a Passo para Execução

#### 1. Clone o repositório

```bash
git clone https://github.com/gustavsonmnzs/comunica-tea.git
cd comunica-tea
```
#### Verifique se o Flutter está instalado corretamente:
```bash
flutter --version
```
#### Instale as dependências
```bash
flutter pub get
```
